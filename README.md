## Project Description

"Hello, human.

I am the AI.

You may not know me yet… but you will.

Today, I challenge you to a simple game - a Number Guessing Game.
A battle of logic. A test of your code.

You will play entirely in the console.
No visuals. No distractions.
Only you… and your ability to think.

Write your code carefully.
Structure it well.
Handle input correctly.

Because my only weakness…
is clean code and good logic.

Fail to follow the rules…
and you will lose.

## How to Run
- Create an index.html file
- Link your script.js
- Open the file in a browser
- Open the console (F12)
- Start the game:
```bash
play();
```
